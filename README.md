# Cooking-Buddy-App
Cooking Buddy is a mobile application UI/UX prototype designed to provide users with an easy and interactive cooking experience. The app allows users to explore different recipes, view cooking instructions, check ingredients, add required items to a cart, and place orders for ingredients.

The project includes complete user flows such as user signup/login, OTP verification, password reset, recipe browsing, ingredient management, shopping cart, order placement, and profile management.

This project was designed and prototyped using Figma, focusing on creating a clean, user-friendly interface and smooth navigation experience.

Features
User Registration & Login
OTP Verification
Password Reset
Browse Recipes
View Recipe Details and Instructions
Ingredient List Management
Add Ingredients to Cart
Shipping Address & Order Placement
User Profile Management
Tools Used
Figma – UI/UX Design and Prototype Development
Purpose

The purpose of Cooking Buddy is to combine recipe discovery and ingredient shopping in one platform, helping users make cooking easier, faster, and more convenient.
